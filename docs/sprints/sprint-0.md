# Sprint 0 - Propuesta inicial

## Objetivo del sprint

Definir la idea inicial del proyecto, el alcance del MVP y las primeras historias de usuario.

## Tareas principales

- Definir historias de usuario iniciales.
- Crear tablero de gestión en Notion.
- Preparar propuesta inicial del proyecto.

## Resultado del sprint

Al finalizar el Sprint 0 se dispone de una propuesta inicial de GymTrack, incluyendo título, descripción, primeras historias de usuario y objetivos del proyecto.

## Estado

Finalizado.
